### Coc commands ... All prepend with ":"
=========================================

- CocInstall, CocUpdateSync, CocUninstall, CocList extensions, CocOpenLog
- `Ctrl-space` = show completions
- `[g` = prev diagnostic
- `]g` = next
- `gi` = implementation
- `gr` = references
- `,rn` = rename

### Go mappings

`,gl` = GoLint
`,gd` = GoDefType
